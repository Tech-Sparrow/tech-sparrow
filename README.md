# tech-sparrow

GitHub Pages site for the [Tech Sparrow](https://github.com/Tech-Sparrow) organisation — organisation hub, repository list (via GitHub API), and connect links.

## Run locally

```bash
python3 -m http.server 8765
```

Open `http://127.0.0.1:8765/`.

## Deploy

Repository **Settings → Pages**: publish from branch `main` and folder `/` (root).
